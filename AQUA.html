Proyecto AQUA

Un sistema para informar sobre el nivel de agua de uno o más reservorios de agua

Un sistema para el sector Acuicola

Este sistema consiste de 1 o varios sensores inalámbricos encargados de medir el nivel de agua

Su funcionamiento consiste en medir la distancia desde el sensor hasta la superficie de agua y lo envía a través de una red propia WiFi hasta un lugar donde haya Internet. luego se puede almacenar estos datos en la nube en tiempo real o se puede utilizar los datos para mostrarlos en una interfaz y activar bombas.

Posee un cable USB para obtener energía, por lo que es compatible con cargadores Móviles o energía fotovoltaica
