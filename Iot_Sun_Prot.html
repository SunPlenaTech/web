IoT Sun Protection

Un sistema para informar a ciudadanos sobre las variables Meteorologicas

Un sistema para Smart Cities

Este sistema consiste de 2 partes esenciales: la Micro Estación Meteorológica (MEM) y la Estación de información Meteorológica (EIM).

Su funcionamiento consiste en que la MEM mide las variables Meteorológicas y lo transmite a una base de datos en la nube y también mediante el protocolo LoRa WAN para distintos usos. En este caso, esa información la reciben las EIM, las cuales permiten visualizar las medidas a través de una pantalla RGB (Multicolor) de una forma rápida. Además, al contar con una conexión WiFi, también permiten al usuario acceder a datos mas detallados desde cualquier dispositivo.

Ambas partes obtienen energía a partir del Sol gracias a sistemas fotovoltaicos que permiten que su funcionamiento sea continuo (MEM) o en horas predefinidas (EIM).

Galería

MEM - MicroEstación Meteorológica

Dispositivo pequeño que mide variables como la Temperatura, Presión Atmosférica, Humedad Relativa, Iluminación y Radiación UV para enviar a una Base de datos integrando tecnologías IoT.

MEM - Sistema Fotovoltaico

Esta solucion obtiene su energía del sol, adecuado para el funcionamiento de la MicroEstacion Meteorológica.

EIM - Estación de información Meteorológica

Te informa los valores meteorológicos en tiempo real.
